# **App Name**: CFDI Explorer

## Core Features:

- XML Import: Import a CFDI 4.0 XML file.
- Data Extraction: Extract data from the XML file and populate a catalog with columns: Version, Folio Fiscal, Serie, Folio, Fecha inicial pago, Fecha final pago, Días pagados, Fecha de pago, Tipo, Clave SAT, Clave interna, Concepto, Importe Gravado, Importe exento, Fecha de Emisión, Fecha de Timbrado, Estatus.
- Catalog Display: Display the extracted data in a tabular format within the application.
- CSV Export: Export the catalog data into a CSV file format.
- XLSX Export: Export the catalog data into an XLSX file format.

## Style Guidelines:

- Primary color: Light sea green (#20B2AA) to evoke trust and stability in financial data management.
- Background color: Very pale sea green (#E0F4F1), a light and unobtrusive background that maintains the airy feel.
- Accent color: Pale cyan (#77DDE7), slightly brighter than the primary color to add vibrancy and to signal interactive elements
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern touch.
- Simple, clear icons for file import and export actions.
- Clean and structured layout for the data table, ensuring easy navigation and readability.
- Subtle animations for loading and data processing states.
